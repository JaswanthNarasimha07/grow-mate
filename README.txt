GrowMate 

GrowMate is a simple plant care tracker that helps users manage their watering schedule. It demonstrates full CRUD functionality users can create new plant entries with light levels and watering frequency, view upcoming watering timelines, update plant names or mark them as watered, and delete plants easily with a swipe or button.


1. Server
cd server
npm install
npm run dev
2. Client
cd client
npm install
npm run dev
Open http://localhost:5173
Data resets when the server stops.
